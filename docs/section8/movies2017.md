# 影视 2017

这一年其实也看了不少影视剧，虽然相比前几年（尤其是在学校的那几年）少了不少。看的少了，一方面是因为没有那么多时间了，一个也是，没有那么多好看的东西了。上映的电影，能让人有看一看的冲动的，寥寥，电视剧更惨淡，于是很多时候只能拿出曾经看过的一些经典，细细再回味。可能我也是老了。
