# Seq2seq

## 经典网络

### Ptr-Net(pointer network)

关于指针网络

Pointer Networks 是发表在机器学习顶级会议 NIPS 2015 上的一篇文章，其作者分别来自 Google Brain 和 UC Berkeley。

指针网络是注意力模型的一种变体，克服了 seq2seq 模型中**输出严重依赖输入**的问题。是最近 seq2seq 比较火的一个分支，在基于深度学习的阅读理解，摘要系统中都被广泛应用。
