                        # Cosine Learning rate decay

                        Learning rate warm up 训练初期由于离目标较远，一般需要选择大的学习率，但是使用过大的学习率容易导致不稳定性。所以可以做一个学习率热身阶段，在开始的时候先使用一个较小的学习率，然后当训练过程稳定的时候再把学习率调回去。比如说在热身阶段，将学习率从 0 调到初始学习率。

                        例如，前面的例如 m 个 batches 用较小的学习率来 warm up（热身）.

                        题外话：对学习率的下降，并非所有人都认同，例如论文 DON’T DECAY THE LEARNING RATE, INCREASE THE BATCH SIZE
